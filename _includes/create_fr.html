Étape 5

Validation du Domaine (TXT)

Puisque nous déservons plusieurs site web sous de multiple domaines, nous devons valider que le domaine vous appartient. Pour ce faire, nous vous demandons d'ajouter un record TXT a votre DNS pour votre domaine. Ceci nous permet de confirmer que le domaine est bien a vous et nous pouvons donc l'associer a votre compte. Il est important de ne jamais effacer ce record.

Note: Remplacez acentera.com par votre nom de domaine a vous.
Par exemple, si vous voulez bâtir : wwww.acentera.com, vous allez devoir créer un record TXT pour acentera-websites.acentera.com. Pour ce faire, vous devez vous connectez chez votre hébergeur de DNS afin d'ajouter le record suivant : acentera-websites.acentera.com TXT "v=xxxxxxxxxxxxxxxxxxxxx". Remplacer la valeur par la valeur affiché lors de la création de votre siteweb. Les double appostrove " sont importants.


Avant de cliquer sur "Validate TXT", vous voudriez peut-être confirmer que votre DNS c'est bien propagé. Voici quelques commandes que vous pouvez utiliser.

Example en utilisant la commande Dig:
dig -t txt +short acentera-websites.acentera.com
"v=c4ca4238a0b923820dcc509a6fXXXXXX"

Example avec nslookup:
nslookup -type=TXT acentera-websites.acentera.com
Vous devriez voir une valeur du genre "v=c4ca4238a0b923820dcc509a6fXXXXXX"

Note: Dans votre cas, vous devriez avoir une valeur différente pour votre siteweb. Chaque compte a un hash unique.

Étape 6

Changement du Domaine (CNAME)

Lorsque le record TXT a bien été validé, nous avons pré-configuré votre site web et nous sommes prêts a héberger votre nouveau site web HTTP2 sécurisé avec votre propre nom de domaine.

Vous allez devoir vous connectez chez votre hébergeur de DNS, et ajouter un CNAME vers acentera-websites.acentera.com tel qu'afficher dans l'image suivant. Remplacer le TestWeb.web.acentera.com par votre propre domaine.



Dépendament de votre configuration, cette étape pourrais prendre un certain temps.
Avant de cliquer sur "Validate DNS Record", vous voudriez probablemetn confirmer le changements a l'aide des outils dig ou nslookup.

Lorsque le tout est prêt, vous devez simplemetn cliquer sur "Validate DNS Record" et nous allons créez le certificats SSL pour votre nom de domaine.
